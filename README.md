1. User memasukkan:
   -berat paket (kg)
   -jarak pengiriman (km)
   -panjang, lebar, tinggi paket(cm)

2. Program menghitung volume = panjang × lebar × tinggi

3. Menentukan tarif per kg berdasarkan jarak:
   ≤ 10 km → Rp 4250/kg
   10 km → Rp 6000/kg
   
4. Tambah Rp 50.000 jika volume lebih dari 100 cm³

5. Menampilkan total biaya
